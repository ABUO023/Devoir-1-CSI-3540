# CSI3540 — Devoir 1  
Création et hébergement d’une page web personnelle

## Hébergement

J’ai choisi **GitHub Pages** comme serveur d’hébergement, car il permet de publier gratuitement des sites web statiques écrits en HTML. Cette solution est simple à utiliser, fiable, et bien adaptée aux projets académiques de petite taille.

## Technologies utilisées

- **HTML** — pour la structure et le contenu de la page web  
- **Tailwind CSS (via CDN)** — pour la mise en forme et le design, sans utiliser de framework JavaScript
